如何使用：</br>
一、部署存龙网：</br>
  1、Linux部署：</br>
    a、登录Linux服务器并创建一个目录</br>
    b、将“Linux部署包”上传至你创建的目录中并解压</br>
    c、为执行程序赋予权限(chmod +x TheIsleLinux)</br>
    d、在SSH控制台cd进入你所创建的目录中并执行./TheIsleLinux</br>
    (首次启动将自动初始化信息并将超级管理员信息提示在控制台窗口，注意查看)</br></br>

  2、Windows部署：</br>
    a、登录Windows服务器并在磁盘中创建一个目录</br>
    b、将“Windows部署包”上传至你创建的目录中并解压</br>
    c、双击运行TheIsleWin.exe</br></br>

  3、如何配置：</br>
    a、conf下的app.conf为存龙网配置文件</br>

二、部署服务器</br>
  1、将项目中“服务器部署文件”上传到游戏服务器根目录并解压</br>
  2、运行“RemotePr.exe”、Run.bat</br></br>

三、部署客户端</br>
  1、将项目中“客户端部署文件”放到游戏根目录并解压</br></br>

详细配置：</br>
“服务器部署文件\TheIsle\Binaries\Win64\service.json”</br>
“客户端部署文件\TheIsle\Binaries\Win64\version.json”</br>

无法正常运行自行检查配置文件关联！！！</br>
